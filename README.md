## 🎯 How to Play

### Basic Mechanics
1. **Match Runes**: Connect 3 or more identical runes in a row or column
2. **Drag & Drop**: Tap and drag tiles to swap positions
3. **Create Combos**: Chain multiple matches for bonus multipliers
4. **Use Special Runes**: Leverage bombs and other power-ups strategically

### Special Runes
- **🔥 Bombs**: Explode surrounding tiles
- **⚡ Line Clearers**: Clear entire rows or columns
- **🔄 Color Changers**: Transform nearby runes
- **💥 Area Clearers**: Remove tiles in a radius
- **✨ Multipliers**: Boost your score

### Scoring System
- **Base Match**: 100 points per rune
- **Combo Multiplier**: Increases with each consecutive match
- **Special Bonuses**: Extra points for using power-ups
- **Level Completion**: Bonus points for reaching targets
